# Side Quest Week 4

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.

To play via GitHub Pages, visit the link provided on the submission page.

Beginning with the home screen, click either buttons based on your decision. Each button will present a new page with new buttons to choose in your journey.

## Assets

No external assets used. All visuals are generated with p5.js.
